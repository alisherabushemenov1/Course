# User Stories

1. As a user, I want to sign up so that I can create a new account in the app.

2. As a user, I want to log in so that I can access my personal account.

3. As a user, I want to view a home screen so that I can see available items.

4. As a user, I want to view detailed information about an item so that I can understand it better.

5. As a user, I want to save my favorite items so that I can access them later.

6. As a user, I want my favorite items to be saved locally so that they remain after restarting the app.

7. As a user, I want to view data fetched from an external API.

8. As a user, I want to manage my app settings such as theme and notifications.
